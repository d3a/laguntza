# QT Designer

Python 3.10 o inferior

## Instalar
```bash
virtualenv venv -p <python10_home>\python.exe
.\venv\Scripts\activate
python.exe -m pip install --upgrade pip
pip install pyqt6
pip install pyqt6-tools
pyqt6-tools designer
```

## Ejecutar
```bash
pyqt6-tools designer
```
