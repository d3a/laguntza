# Tuplas
Las  _tuplas_ son como los _listas_ pero sus elementos no puden ser modificados.

## Declarar Tuplas
```python
  tupla = tuple()
  tupla = (1, 2, 3, 4)
```
Los elementos pueden ser de cualquier tipo:
```python
  tupla = tuple()
  for i in range(1, 5)
    obj = Objeto()
    tupla.append(obj)
```
